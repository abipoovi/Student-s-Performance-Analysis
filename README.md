# Student Performance Prediction

This project predicts student marks using Machine Learning.

## 📌 Description
A regression model is built using Linear Regression to predict marks based on:
- Study Hours
- Attendance

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📊 Model
- Algorithm: Linear Regression
- Evaluation: Mean Absolute Error

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python main.py

## 📈 Output
- Predicted student marks
- Visualization of study hours vs marks
